# Troubleshooting Common Issues at NexaCorp

## Overview
This document lists common issues encountered by employees while using NexaCorp's internal systems,
along with recommended troubleshooting steps.

## Login & Access Issues
### Unable to Log In
- Verify username and password
- Ensure MFA device is available
- Check if account is locked due to failed attempts

### Access Denied Errors
- Confirm assigned role and department
- Verify that access has been approved
- Check if permissions were recently updated

## VPN & Network Issues
- Ensure VPN client is installed and running
- Reconnect VPN before accessing internal resources
- Avoid public Wi-Fi without VPN

## Application Errors
- Clear browser cache and retry
- Check system status page for outages
- Report persistent errors to IT Support

## Data Access Issues
- Some data may be restricted based on role
- Recently updated policies may not be immediately visible
- Contact data owners for clarification

## Getting Help
- IT Support: it-support@nexacorp.internal
- HR Support: hr@nexacorp.internal
